# DevOps Metrics API

A professional REST API for tracking infrastructure deployment metrics — built as a portfolio project demonstrating backend development skills (FastAPI, PostgreSQL, JWT auth, testing, Docker, CI/CD).

## Status: 🚧 In development

## Planned features
- CRUD for Deployments, Services, Users
- Aggregated metrics endpoint
- JWT authentication
- PostgreSQL + SQLAlchemy + Alembic migrations
- pytest test suite (80%+ coverage target)
- Dockerized (API + PostgreSQL)
- CI/CD via GitHub Actions
- OpenAPI/Swagger docs (auto-generated by FastAPI)

## Tech stack
- Python 3.10
- FastAPI
- PostgreSQL
- SQLAlchemy + Alembic
- Pydantic
- pytest
- Docker / Docker Compose
- GitHub Actions
